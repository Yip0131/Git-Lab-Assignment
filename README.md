Task 7: Reflection
Reflection on Version Control in Software Development
Version control systems such as Git, commonly used with platforms like GitHub, significantly improve both software quality and team collaboration.
1. What I learned
- Basic Git commands (init, add, commit, push, pull, branch, merge)
- How to resolve merge conflicts
- Pull request workflow on GitHub

2. Challenges faced
- Understanding staging area vs working directory
- Resolving first merge conflict manually

3. How Git helps in real projects
- Tracks history and allows rollback
- Enables team collaboration without overwriting work
- Supports code review via pull requests

4. Improved Software Quality
Version control enhances quality through:
* Change Tracking
- Every modification is recorded with a commit history, making it easy to trace errors and understand what changes were made.
* Rollback Capability
- Developers can revert to previous stable versions if bugs or issues occur, reducing risk during development.
* Structured Development
- Features and fixes can be developed in branches, tested, and only merged when verified, ensuring stability of the main codebase.

5. Better Teamwork and Collaboration
* Parallel Development
- Multiple developers can work on different features simultaneously using branches without interfering with each other’s work.
* Code Review Process
- Tools like Pull Requests allow team members to review, comment, and improve code before merging, leading to higher-quality output.
* Conflict Management
- Version control systems detect conflicts when changes overlap and provide mechanisms to resolve them systematically.
* Transparency and Accountability
- Each commit records the author and changes made, improving accountability and project tracking.

Reflection answer:
Version control (Git) improves quality by maintaining history, enabling experimentation via branches, preventing code loss, and supporting parallel work. Teamwork improves because members can review each other’s code via PRs, resolve conflicts systematically, and integrate features safely.
